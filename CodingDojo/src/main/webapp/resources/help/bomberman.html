Bombarman Coding dojo — how to play?

That's easy! First of all you have to run playserver somewhere in LAN. Every participant opens playserver page in browser http://server_ip:8080

On the page Read help you may download the bot template and read the rules

There are clients for different platforms, all of them are in zip file. Download it and open in your favorite IDE

First of all you have to register your bot in playserver

After this server will start to recognize you

In any moment you may leave the game by choosing Unregister

To start you have to run your bot template. Look at README file in the root folder.

Class that implements AI is named DirectionSolver. What you may return from method DirectionSolver.get()? Only 5 commands, all of them are described in enum Direction: UP, DOWN, LEFT и RIGHT — movement directions, а ACT - set the bomb

Constructor of DirectionSolver has one argument - board — it's an object that represents playing board. On the board you may see few types of objects. Their description is in enum Elements.

Ok, let's go back to main menu

Here you may look at game boards

Runtime game info
- Player Name
- Number of player points
- Maximum points gathered for one bot live
- Level that you're on (TODO: not changing yet)

You're getting points for blowing up different objects on board and getting penalty when your bot dies

Have a nice game!
